\doxysubsection{Descripció Detallada}
Representa un virus amb totes les seves propietats i característiques epidemiològiques. 

Aquesta classe serveix com a estructura de dades temporal per emmagatzemar la informació d\textquotesingle{}un virus llegit des d\textquotesingle{}un fitxer.

És una classe de dades utilitzada per transportar informació entre les capes de l\textquotesingle{}aplicació.
